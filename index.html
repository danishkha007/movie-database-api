üöÄ GitHub Pages Deployment

Step 1: Repository Setup

  1. Create a new GitHub repository
  2. Upload the three files: index.html, style.css, and app.js
  3. Go to repository Settings ‚Üí Pages
  4. Select "Deploy from a branch" and choose "main" branch

Step 2: Custom Domain (Optional)

  1. Add a CNAME file to your repository with your domain name
  2. Configure your DNS provider to point to GitHub Pages:
    • CNAME record: your-domain.com ‚Üí username.github.io
    • Or A records pointing to GitHub's IPs
  3. Enable "Enforce HTTPS" in repository settings

Step 3: Using the API

Once deployed, your API will be accessible at:

  • GitHub Pages: https://username.github.io/repository-name/#/api/movies
  • Custom Domain: https://your-domain.com/#/api/movies

Note: This is a client-side API simulation. All data is static and changes won't persist.
